# C++ 2D Function Approximation

Реализована программа для приближения функций двух переменных с помощью билинейной интерполяции и двумерного кубического сплайна. Программа вычисляет значения выбранной функции в узлах прямоугольной сетки, строит приближённую поверхность и отображает график абсолютной погрешности с помощью Gnuplot.

## Возможности

* выбор прямоугольной области вычислений;
* настройка количества узлов по осям `x` и `y`;
* вычисление значений функции в узлах сетки;
* билинейная интерполяция функции;
* построение двумерного кубического сплайна;
* решение трёхдиагональной системы методом прогонки;
* вычисление абсолютной погрешности приближения;
* построение трёхмерных поверхностей в Gnuplot.

## Структура проекта

* `main.cpp` — точка запуска программы, обработка аргументов и создание сетки;
* `linear_aprox.cpp` — реализация билинейной интерполяции;
* `linear_plot.cpp` — построение поверхности билинейной интерполяции и её погрешности;
* `cubic_spline_aprox.cpp` — реализация двумерной сплайн-интерполяции;
* `cubic_spline_plot.cpp` — построение поверхности кубического сплайна и графика его погрешности;
* `utility.cpp` — исходные функции, вторые производные, одномерный кубический сплайн и метод прогонки;
* `Header.h` — объявления функций и вспомогательные определения.

## Поддерживаемые функции

Параметр `k` определяет функцию, которую необходимо приблизить:

* `0` — (f(x,y)=1);
* `1` — (f(x,y)=x);
* `2` — (f(x,y)=y);
* `3` — (f(x,y)=x+y);
* `4` — (f(x,y)=\sqrt{x^2+y^2});
* `5` — (f(x,y)=x^2+y^2);
* `6` — (f(x,y)=e^{x^2-y^2});
* `7` — (f(x,y)=\frac{1}{25(x^2+y^2)+1}).

## Компиляция

С помощью `g++`:

```bash
g++ main.cpp cubic_spline_aprox.cpp cubic_spline_plot.cpp linear_aprox.cpp linear_plot.cpp utility.cpp -o approximation_2d
```

При использовании исходных названий файлов с добавленными номерами команда может выглядеть так:

```bash
g++ "main(2).cpp" cubic_spline_aprox.cpp cubic_spline_plot.cpp linear_aprox.cpp linear_plot.cpp "utility(1).cpp" -o approximation_2d
```

Перед компиляцией необходимо убедиться, что подключаемый заголовочный файл называется `Header.h`. При необходимости файл `Header(2).h` следует переименовать:

```bash
mv "Header(2).h" Header.h
```

## Запуск

Программа принимает девять аргументов:

```bash
./approximation_2d <ax> <bx> <ay> <by> <nx> <ny> <mx> <my> <k>
```

Где:

* `ax`, `bx` — границы области по оси `x`;
* `ay`, `by` — границы области по оси `y`;
* `nx`, `ny` — количество узлов по осям;
* `mx`, `my` — дополнительные параметры сетки;
* `k` — номер приближаемой функции.

Пример запуска:

```bash
./approximation_2d -1 1 -1 1 10 10 100 100 7
```

В данном примере:

* используется область ([-1,1]\times[-1,1]);
* создаётся сетка размером `10 × 10`;
* приближается функция
  (f(x,y)=\frac{1}{25(x^2+y^2)+1}).

## Принцип работы

Программа создаёт равномерную прямоугольную сетку и вычисляет значения выбранной функции во всех её узлах.

При билинейной интерполяции точка находится внутри одной из ячеек сетки, после чего её значение вычисляется по четырём соседним узлам.

Двумерный кубический сплайн строится последовательно. Сначала одномерные сплайны вычисляются вдоль оси `x`, а затем полученные значения интерполируются вдоль оси `y`. Для определения коэффициентов одномерных сплайнов решается трёхдиагональная система.

После вычисления приближения программа строит:

* поверхность приближённой функции;
* поверхность абсолютной погрешности (|f(x,y)-S(x,y)|).
